Usage Guide
1. 在 Start Simulation 设置参数后点击 Start。端口可留空，系统会自动分配。
2. 点击 Refresh 或等待自动刷新，在 Scanned Sim Processes 查看进程、ZMQ 端口、队伍人数与 WebView 链接。
3. 在 Process Control 选择 PID 后，点击 Stop PID 可停止单个仿真；点击 Stop All 可停止全部仿真。
4. 选择 PID 后展开 Decider Commands，可复制每个机器人的 decider 启动命令。
5. 页面底部 Logs 会显示启动和停止操作结果。
