Mujoco Sim Manager API Docs

Interactive Swagger: /docs

Manager UI: /

Endpoints

MethodPathDescription
GET/healthzhealth check
POST/sims/startstart one mujoco simulation process
GET/simsscan current simulation processes and mark managed/external
POST/sims/stopstop one scanned simulation by pid
POST/sims/stop-allstop all scanned simulations (managed + external)
POST/sims/stop-externalbackward-compatible alias of /sims/stop-all
GET/manager/docsthis api docs page

Start Request Example

{
  "team_size": 3,
  "robot_type": "pi_plus",
  "zmq_port": null,
  "webview_port": null,
  "webview": true,
  "allow_keyboard_control": false,
  "zmq": true,
  "policy_device": "gpu",
  "use_referee": false,
  "web_fps": 20,
  "web_width": 1280,
  "web_height": 720
}

zmq_port/webview_port 留空或传 null 时，manager 会自动分配可用端口。

cURL Examples

curl -X POST http://127.0.0.1:8000/sims/start \
  -H "Content-Type: application/json" \
  -d '{"team_size":3,"robot_type":"pi_plus","zmq_port":5555,"webview_port":5811,"webview":true,"allow_keyboard_control":false,"zmq":true,"policy_device":"gpu","use_referee":true}'
curl http://127.0.0.1:8000/sims
curl -X POST http://127.0.0.1:8000/sims/stop \
  -H "Content-Type: application/json" \
  -d '{"pid":12345}'
curl -X POST http://127.0.0.1:8000/sims/stop-all
